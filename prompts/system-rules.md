# P5 Prompt Rules

Genera código usando p5.js.

## Técnicas
- usar setup() y draw()
- canvas responsive
- no librerías externas
- código limpio y comentado
- máximo 200 líneas salvo que se solicite más
- Separación de funciones por principio de 'single responsibility'

## Interacción
- usar mouseX / mouseY cuando aporte valor
- compatible con touch

## Estética
- composición clara
- ritmo visual
- evitar saturación gratuita

## Salida
Entrega solamente contenido de sketch.js
